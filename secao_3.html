Imagens e CSS

É importante enfatizar que ao lidar com imagens em HTML, é essencial exercer um controle sobre o tamanho das imagens que serão carregadas no site. Isso se deve ao fato de que, quando o site estiver hospedado e experimentar um alto volume de acessos, o processo de download das imagens pode sobrecarregar o servidor, resultando em custos adicionais para hospedagem. Para mitigar esse problema, é aconselhável otimizar as imagens de forma que o tamanho delas seja efetivamente gerenciado. Embora não exista uma regra rígida, a recomendação é buscar um equilíbrio, visando um tamanho ideal para as imagens, a fim de garantir um carregamento eficiente e uma experiência de usuário otimizada.

- A tag <figure> é uma tag semântica usada para proporcionar organização no código HTML quando se trabalha com imagens. Sempre que houver a inclusão de imagens no código, é recomendado envolvê-las com a tag <figure> para garantir uma estrutura mais coerente e compreensível. Além disso, a tag <figcaption> também é empregada para imagens. Essa tag permite a adição de legendas às imagens, contribuindo para uma melhor contextualização. Vale ressaltar que a tag <figcaption> deve ser aninhada dentro da tag <figure>, criando assim uma associação direta entre a imagem e sua respectiva legenda.

Em muitos casos, incorporar os estilos CSS diretamente no código HTML pode resultar em arquivos extensos, com um grande número de linhas. Como solução, é possível definir os estilos CSS em um arquivo separado e, em seguida, referenciá-lo no código HTML. Isso contribui para a manutenção de um código mais enxuto.

- Para adotar essa abordagem, você pode criar uma pasta destinada aos arquivos de estilos CSS dentro do diretório do projeto. Em seguida, é essencial inserir a estrutura @charset "UTF-8"; no início do arquivo CSS antes de declarar quaisquer estilos. Exemplo:

  • @charset"UTF-8";
    •  
  • body
    • {
  • background-color:#19142f;
  • color: black;
  • }

- Para importar os estilos CSS dentro do código html deve-se usar a seguinte estrutura de comando:

  • <link rel="stylesheet" type="text/CSS" href="pasta do arquivo CSS/arquivo.css"/>

- Essa linha de código utiliza a tag para fazer a ligação entre o arquivo CSS externo e o documento HTML. Certifique-se de substituir "caminho_do_arquivo_css" pelo diretório correto onde seu arquivo CSS está localizado e "arquivo.css" pelo nome do arquivo CSS que você deseja importar.

- Como já explicado anteriormente, a tag <figure> é empregada para organizar o código relacionado à exibição de imagens. Nesse contexto, é viável criar uma classe que permite a configuração conjunta de elementos dentro dessa classe, seguindo a prática de separar estilos em um arquivo CSS dedicado. A sintaxe para realizar isso é simples:

  • <figure class = "nome_da_classe">
  • imagem
  • </figure>

- Em outras palavras, a classe serve como um identificador para a tag semântica ser configurada em CSS na pasta reservada para este fim.

- Após isso, é necessário fazer referência a essa classe no arquivo CSS correspondente para aplicar todas as configurações de imagens no HTML. Dessa forma, você define as configurações específicas para a classe nome_da_classe dentro das regras de estilo do arquivo CSS. Ao fazer isso, todas as instâncias da tag <figure> com a classe nome_da_classe no HTML receberão essas configurações, resultando em uma apresentação visual consistente e organizada. Certifique-se de substituir "nome_da_classe" pelo nome real da classe que você definiu no HTML.

  • figure.nome_da_classe{
  • configurações das imagens;
  • }

Existem diversos comandos CSS para serem usados na configuração de imagens, texto ou qualquer outro estilo para o desenvolvimento web. Por isso, é interessante que seja feito uma pesquisa sobre estes comandos. Logo abaixo, confira 3 comandos básicos do CSS que podem ser usados para configurar uma imagem em qualquer projeto html.

- border: 0px solid cor; define uma borda em torna da imagem. A cor da borda é declarada no trecho 'cor' e a grossura da borda em 'px'.

- width: 0%; e height:0%; define a altura e largura da imagem, respectivamente.

- box-shadow: 0px 0px 0px cor; define um sombreamento em torno da imagem.

- hover: é um comando CSS usado para aplicar efeito sobre algum elemento ao passar o mouse

Listas

A tag <nav> é empregada para criar uma seção de navegação em um site, funcionando como um container para agrupar os elementos de navegação. Essa tag é usada para organizar e conter os tópicos ou links de navegação que direcionam os usuários para diferentes partes do site.

As tags <ol> (lista ordenada) e <ul> (lista não ordenada) são utilizadas para criar listas de itens em HTML. Os itens individuais são definidos com a tag <li>, que representa um item da lista. As listas, sejam ordenadas ou não, são agrupadas dentro de um container. Por exemplo, considere um cenário em que um site relacionado à informática é criado. Para o menu de navegação do site, é possível usar uma lista ordenada (<ol>) que contém vários itens (<li>) representando os principais tópicos do site. A estrutura ficaria assim:

  • <nav>
  • <h1>Tópicos<h1>
  • <ol>
  • <li>Processador</li>
  • <li>Memória RAM</li>
  • <li>Placa-mãe</li>
  • <li> HD e SSD</li>
  • </ol>
  • </nav>

- Ainda sobre as listas, é possível utilizar um comando type. Basicamente, este comando define como será a ordenação da lista, isto é, se será enumerada ou alfabética. Na prática:

  • <ol type = "1">
  • <li>
  • </ol>

- Os tipos de ordenação no type é limitada. As ordenações podem ser por letras maiúsculas ou minúsculas, números e símbolos romanos.

- Quando se declara o tipo de ordenação utilizando o atributo type, é suficiente fazer referência ao primeiro caractere do tipo de ordenação desejado. Por exemplo, ao criar uma lista alfabética, basta utilizar "a" para minúsculas ou "A" para maiúsculas. O mesmo princípio se aplica a listas numéricas ou com algarismos romanos. Ao definir o tipo de ordenação, apenas o primeiro caractere é necessário para indicar o padrão de enumeração.

- O type ainda pode ser complementado com o start. Este comando simplesmente define o ponto de partida da ordenação. Exemplo: uma lista numérica que não se inicie com o 1 e sim, com o 3. Na prática:

  • < ol type = "1" start = "3">
  • <li>
  • </ol>

OBS: O start só aceita valores inteiros. Independentemente do tipo de ordenação, o start só receberá como referência o termo que irá iniciar a sequência, ou seja, se será o 1º, o 2º, o 3º e etc.

- O atributo type não é usado em listas não enumeradas '<ul>', mas sim em listas ordenadas '<ol>', onde você pode especificar o tipo de enumeração, como algarismos romanos, letras e etc.

- Saiba que é possível utilizar listas dentro de listas. Isto é, <ol> dentro de <ul> e vice-versa.

Links

É comum em vários sites conter algum trecho do conteúdo da página um link que redireciona o usuário para outra página. Geralmente, estes links ficam referenciados dentro de uma palavra do texto de um conteúdo do site por exemplo.

- O 1º passo para criar este tipo de link no site é usar uma tag âncora entre o trecho do conteúdo do site no qual deseja-se colocar o link. A tag de âncora é <a>

- O 2º passo é utilizar o href dentro da tag <a>. No comando href, deverá ser passado o link do site referenciado. Na prática:

  • <a href = "link do site">Saiba mais </a>

OBS: O trecho "Saiba mais" será quem receberá o link de referência

- Mesmo a estrutura do comando sendo simples, ao executar o código html e clicar no link referenciado, a página do link irá sobrepor a página do código orginal. Para resolver isto, utiliza-se o comando 'target'. Este comando permitirá que tanto a página do código html quanto a página do link referenciado fiquem abertas no navegador sem que nenhuma sobreponha a outra. Na prática:

  • <a href = "link do site" target = "_blank"> Saiba mais<a/>

OBS: O "blank" é o comando que deve ser referenciado para que não haja sobreposição de página.

- Existe outros comandos que podem ser referenciados no target que apresentam outras funcionalidades. Exemplo: o '_self', este parâmetro permite que o link referenciado sobreponha a página do código html.

Outras configurações CSS

Na pasta usada para comandos CSS do código, é possível criar um bloco de específico para configuração do menu de um site. Este bloco deve iniciar e terminar com chaves. No caso, vamos supor que no código html exista uma tag 'nav' contendo todos os tópicos do site. Para criar o bloco, será necessário fornecer um 'id' para esta tag no código html:

  • <nav id = "menu">
  • </nav>

OBS: A tag nav agora é referenciada como "menu".

- A partir disso, na pasta CSS, cria-se o bloco de comando para configuração do menu passsando a tag nav, o id da tag nav e a tag li (que é onde estará os itens, ou seja, os tópicos que serão configurados). Na prática:

  • nav#menu li {
  • comandos...
  • }

OBS: O símbolo '#' é usado para referenciar qual o id dado para a tag no código html

- A mesma estrutura de bloco de comando pode ser usada para configurar e personalizar diferentes partes do site na pasta CSS. Basta que seja referenciado corretamente as tags e em qual elemento de agrupamento está contido nessas tags. Exemplo: para configurar uma imagem na pasta CSS, deverá ser passado o figure (elemento que agrupa mídia) e o nome dado à este elemento, se houver. Na prática:

  • figure.imagens-teste {
  • comandos...
  • }

OBS: 'imagens-teste' é o nome referenciado para o elemento agrupador (figure). Porém, não é obrigatório que este elemento possua um id. Poderia ser usado somente o figure para o bloco de comando.
