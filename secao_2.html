Estrutura Básica do HTML

Partindo para as codificações, é importante mencionar, antes de mais nada, que todo documento HTML começa com uma tag de abertura e de fechamento. Uma tag é um elemento utilizado pela linguagem para definir o conteúdo do site que está sendo criado. No caso, a primeira tag a ser escrita é a que envolverá todo o código HTML; em outras palavras, é a tag que marca o início do que será desenvolvido. Na prática:

  • <html>
  • espaço onde será desenvolvido o documento html
  • </html>

- Note que, na tag de fechamento, é utilizada uma barra dentro dos sinais de menor e maior (colchetes angulares). Essa barra indica o fechamento do conteúdo. Toda tag que encerra algum conteúdo em HTML contém essa barra.

Toda tag deve ser escrita dentro de sinais de menor e maior '<>'.

Antes de iniciar a estrutura de comandos HTML, é necessário declarar a instrução <!DOCTYPE html>. Essa instrução informa ao navegador que o código que será desenvolvido é um documento HTML. O navegador utilizado para executar o código pode ser qualquer um. Na prática:

  • <!DOCTYPE html>
  • < HTML lang="pt-br">
  • </html>

OBS: O comando pt-br" informa o idioma no qual será escrito o documento HTML.

Existem outras duas tags importantes que formam a estrutura básica do código HTML. Essas tags são <head> e <body>. Na primeira tag, estarão contidos os comandos responsáveis pelo comportamento e pelas configurações do site. Na segunda tag, estará contido o conteúdo visual do site, como títulos, cores etc. Na prática

  • <!DOCTYPE html>
  • <html lang = "pt-br">
  • <head>
  • <meta charset = "UTF-8"/>
  • <title> Nome do Site </title>
  • </head>
  • <body>
  • <h1> Olá, mundo</h1>
  • <h2> Maldição retirada </h2>
  • </body>
  • </html>

- Note que na tag <head> estão os comandos que realizarão a configuração do site. O comando <meta charset="UTF-8"> será utilizado para definir o conjunto de caracteres como UTF-8, permitindo a exibição correta de caracteres especiais. Além disso, o segundo comando, <title>, definirá o título do site que será exibido na barra de título do navegador.

- Dentro da tag <body>, estão os comandos que definirão a parte visual do site. No caso, o que será exibido é "Olá, mundo" e "Maldição retirada". Esses textos estão envoltos na tag <h1>, que é usada para definir títulos principais do conteúdo da página. Em seguida, a tag <h2> é utilizada para criar sub-títulos relacionados ao conteúdo. Da mesma forma, a tag <h3> pode ser utilizada para criar sub-títulos dos sub-títulos. A lógica segue com as tags <h4>, <h5>, <h6> e assim por diante, para hierarquias de títulos em ordem decrescente.

CSS

O atributo style é utilizado para definir estilos de CSS diretamente no documento HTML. O style permite a inclusão de estilos da tecnologia CSS dentro do HTML, o que significa que ele adota a sintaxe do CSS para aplicar estilos aos elementos HTML.

- O atributo style é responsável por definir o estilo visual do conteúdo do site, como o tipo de fonte, tamanho da fonte, cores, entre outros aspectos de apresentação. Na prática:

  • <style>
  • h1{
  • font-family: Arial;
  • font-size: 30pt;
  • color: black;
  • }
  • </style>

A tag span é usada para aplicar estilos CSS no texto ou em pequenos trechos do texto. Na prática:

  • <span style = " ">texto</span>

- Dentro das aspas, é onde você pode fornecer os parâmetros de estilo para a formatação do texto. Esses parâmetros podem incluir propriedades como font, color e muitos outros do CSS para personalizar a aparência do texto contido na tag <span>. É importante notar que os parâmetros e seus valores devem seguir a sintaxe e as regras do CSS para que a formatação seja aplicada corretamente.

Para centralizar textos ou até mesmo definir uma margen para um paragráfo em um texto, é usado o atributo style com os seguintes comandos:

  • <style>
  • p{
  • text-align:center;
  • }
  • </style>

- Esta estrutura será a responsável por centralizar o texto que estiver dentro da tag 'p'.

- Logo em seguida, a outra estrutura de comando abaixo, será a responsável por justificar o texto e ainda, adicionar uma margen de um parágrafo no texto (no caso, a margen do parágrafo no comando é de 50px). Lembrando que o style deve ser declarado na tag comportamental do documento html, ou seja, na tag head.

  • <style>
  • p {
  • text-align: justify;
  • text-indent: 50px;
  • }

- Desta forma, toda vez que for usada a tag p já será gerado o parágrafo no texto.

Comandos HTML

<img src = "nome do documento de imagem.formato">. Esta é uma tag usada para adicionar imagens no código html. Dentro das aspas do comando é necessaário passar o nome da imagem e o formato de arquivo desta imagem (jpeg, png, gif e etc). Ainda, para facilitar o uso desta tag, é interessante salvar a imagem dentro do mesmo repositório do código html. Na prática:

  • <img src = "imagem-exemplo.JPG"/>

A tag <br> é usada para a quebra de linhas dentro do texto ou entre títulos no código HTML. Outro comando similar que pode ser usado em conjunto é o nbsp;. Este comando é usado para criar espaços entre textos. Sua aplicação correta no código deve ser:

  • <h1> texto 1 &nbsp texto 2 <h1>

- Estes comandos de formatação são importantes em qualquer texto dentro do código html. Mesmo que esteja todo organizado no código html, o texto será mostrado de forma desorganizada e confusa no navegador. Isso acontece porque o html não reconhece espaços, parágrafos e quebra de linhas se não por meio de tags específicas como as apresentadas para esta finalidade.

<p> é usada para criar parágrafos. Se dado um texto, este estiver dentro desta tag, ao executar o código no navegador, será mostrado o texto em um parágrafo.

<wbr> tag usada para quebra de palavras no texto. Este é um comando usado para para permitir que uma palavra não estrague o layout da página no navegador.

A tag <del> é usada para riscar um texto, indicando que ele foi apagado

Dentro da tag head, é possível utilizar uma estrutura de comandos para definir cores para o site. Na prática:

  • body{
  • background-color:#a514cd;
  • color: black;
  • }

- O background-color define a cor de fundo do elemento. O color define a cor do texto dentro do elemento. O código de cores declarado no background-color é uma representação hexadecimal que combina três canais de cores: vermelho, verde e azul. Cada canal vai de 00 (nenhuma intensidade) a FF (intensidade máxima). Essa combinação cria uma ampla gama de cores.

  a5: indica a quantidade de vermelho
  14: quantidade de verde
  cd: quantidade de azul

Importante mencionar que ao invés do código hexadecimal, poderia ser simplesmente declarado o nome de uma cor para todo o site. Porém, com este código, é possível ter uma ampla gama de cores e variações de cores.

Para gerar códigos hexadecimais para as cores é possível utilizar ferramentas on-line que fornecem seletores de cores e geram códigos hexadecimais automaticamente, usar extensões ou plugins de navegadores que permitem capturar cores diretamente das páginas da web e explorar bibliotecas de cores em frameworks de design para encontrar paletas adequadas.

Outra forma de configurar cores com o background é com o rgb(). Neste comando, os valores de tonalidade de cada uma das cores é passado sem usar números hexadecimais, ou seja, de forma inteira. Na prática:

  • background-color: rgb(85,100,150);
