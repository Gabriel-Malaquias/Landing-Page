HTML e CSS

O que é HTML ?

HTML é uma linguagem usada para a criação de páginas web. Essa linguagem é utilizada em conjunto com o protocolo HTTP (Hypertext Transfer Protocol), o qual é empregado para a transferência de diversos tipos de recursos, como imagens, vídeos, músicas etc., através da rede de computadores. Em resumo, o HTML é responsável por criar o conteúdo das páginas web, enquanto o HTTP é o protocolo que possibilita a transferência dessas páginas pela rede. Através desse processo de transferência, é possível acessar um site, por exemplo.

World Wide Web (www) é o que engloba todos os servidores que utilizam o HTTP.

Os navegadores desempenham o papel de permitir que os usuários acessem sites e visualizem suas páginas web. Eles realizam essa função através da comunicação com os servidores web utilizando o protocolo HTTP, que conecta o computador ou celular do usuário ao servidor que armazena o documento HTML ou site requisitado.

E o CSS ?

O HTML não atua isoladamente na construção de sites. Ele é complementado pela linguagem CSS (Cascading Style Sheets).

O CSS é a linguagem que lida com o estilo do site, abrangendo aspectos como cores, fontes, tamanhos e posicionamento de elementos visuais, como botões. Basicamente, o CSS é responsável por definir a aparência estética do site, enquanto o HTML define a estrutura dos elementos presentes na página, como parágrafos, links, títulos, imagens, vídeos, entre outros.

Informações complementares

Ao criar um site, é necessário realizar sua hospedagem. A hospedagem de site é um processo que envolve o armazenamento do site em um servidor web, permitindo sua publicação e acesso por parte dos usuários da internet.

É importante destacar que sem a hospedagem, o site não estará disponível na internet e, consequentemente, não poderá ser acessado pelos usuários. Os serviços de hospedagem geralmente oferecem recursos como backups, bancos de dados, suporte técnico, armazenamento de arquivos, entre outros. A escolha do provedor de hospedagem deve ser feita com cautela, uma vez que isso influencia na eficiência do site e na qualidade do serviço. Geralmente, a hospedagem de site é um serviço pago. Embora existam opções de hospedagem gratuita, estas costumam oferecer recursos computacionais limitados ao site.

Além da hospedagem, um site também deve possuir um domínio. O domínio consiste no endereço ou nome do site, juntamente com a sua extensão de domínio. O domínio deve estar relacionado ao tipo ou propósito do site. Por exemplo, um domínio como "vendas" pode indicar um site relacionado a atividades de comércio. A extensão de domínio é a parte final do endereço web e desempenha um papel importante na identificação do propósito do site. Aqui estão algumas das principais extensões de domínio mais utilizadas:

  • .com: extensão usada para um site global. Geralmente usada em empresas comerciais
  • .org: extensão usada em sites de empresas sem fins lucrativos
  • .edu: extensão usada em sites de instituições educacionais
  • .gov: extensão usada para sites governamentais
  • .br: extesão usada para determinar a localização geográfica de um site. No caso, a extensão br indica sites, endereços de e-mail relacionados ao Brasil. Cada país pode ter sua própria extensão de domínio. A extensão de domínio também é um serviço que pode ser oferecido por um provedor de hospedagem
