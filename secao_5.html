JavaScript

Para iniciar a explicação, é importante ressaltar que JavaScript não tem nada a ver com a linguagem Java em si. O nome "Java" em "JavaScript" surgiu devido a uma jogada de marketing entre duas empresas: Netscape e Sun Microsystems.

- A Sun Microsystems era a empresa que estava lançando a linguagem de programação Java, enquanto a Netscape estava desenvolvendo a linguagem LiveScript. Por meio de uma parceria entre essas duas empresas, a Netscape decidiu incluir "Java" no nome de sua linguagem como uma estratégia de marketing. Assim, o JavaScript foi criado.

- Apesar de não terem uma relação direta, as linguagens Java e JavaScript possuem semelhanças em sua sintaxe, uma vez que ambas são baseadas na linguagem C.

- O objetivo principal do JavaScript é proporcionar a interatividade do usuário com uma aplicação web. Em outras palavras, essa linguagem permite que o usuário interaja com uma página da web, como preencher um formulário ou usar botões de clique, por exemplo.

Todo código de JavaScript deve ser incluído dentro da tag <script></script>.

O comando document.write(""); é o comando usado para escrever alguma mensagem no corpo do site.

A função alert(""); é usada para exibir uma caixa de alerta na página. Esta caixa é acompanhada por um botão "OK" para fechá-la.

OBS 1: Em JavaScript, a maioria dos comandos deve ser finalizada com um ponto e vírgula (;) para indicar o término de uma instrução. É uma boa prática sempre incluir ponto e vírgula para evitar possíveis erros de interpretação do código pelo navegador. No entanto, o JavaScript também possui um mecanismo chamado "inserção automática de ponto e vírgula" (automatic semicolon insertion) que adicionará automaticamente pontos e vírgulas em certos casos onde eles estão ausentes.

OBS 2: É totalmente possível que haja mais de um script no html.

Funções

Em JavaScript para criar uma função deve-se seguir a seguinte estrutura:

  • function nomeFuncao() {}

- Lembrando que todo comando JavaScript deve estar dentro de <script>

- Na função, o que estiver dentro dos parênteses são os parâmetros da função. Enquanto que, o que estiver dentro das chaves são os blocos de códigos.

O atributo onclick é usado para associar uma função JavaScript a um evento de clique em um elemento HTML, como uma imagem, botão ou qualquer outro elemento interativo.

O comando onmousemove tem a mesma ideia que o onclick. A diferença é que o usuário só precisará passar o mouse por cima do elemento para ser feita uma ação. No caso da imagem, quando o mouse passar pelo elemento uma ação será feita.

O comando onmouseout faz o oposto do onmousemove já que, irá chamar uma função quando o mouse for tirado do elemento.

OBS: Parâmetros são valores que são passados do seu código principal para uma função.

No JavaScript, para criar condicionais a sintaxe é a seguinte:

  • if(condição) {
  • código;
  • }

Da mesma forma que é possível carregar comandos CSS de outra pasta no código html, também é possível fazer o mesmo com o JavaScript. Para isso, basta criar uma pasta no repositório html que irá guardar os comandos JavaScript.

OBS: A pasta dos comandos JavaScript deve conter a extensão js.

- Para importar arquivos JavaScript de uma pasta no seu código HTML, você deve usar a tag <script> e o atributo src para apontar para o caminho relativo do arquivo JavaScript que você deseja incluir. Essa tag pode ser colocada tanto na seção <head> quanto no final do corpo (<body>) do seu arquivo HTML.

- A estrutura de comando usada para a importação, é a seguinte:

  • <script>
  • src = "pasta_JavaScript/arquivo.js"
  • </script>

Criando Formulário com JavaScript

Para criar formulários no código HTML, primeiramente, é necessário inserir todo o conteúdo do formulário no código. Este conteúdo deve estar inserido dentro da tag chamada <form></form> (tag utilizada para criar formulários).

- Cada formulário possui conjuntos de campos que identificam diferentes partes do mesmo. Por exemplo, em um formulário existem conjuntos de campos que solicitam informações básicas de uma pessoa, como nome, telefone, CPF, entre outros, assim como existem conjuntos de campos que pedem informações sobre o endereço, como logradouro, número da casa, rua, etc. Em outras palavras, esses conjuntos de campos representam diferentes grupos do formulário, cada um solicitando informações distintas.

- Cada um dos conjuntos de campos de um formulário deve ser inserido dentro da tag <fieldset></fieldset>. Isso significa que dentro da tag <form> pode haver vários <fieldset>.

- <legend></legend> é uma tag usada para inserir os títulos dos conjuntos de campos do formulário. Ela é utilizada dentro da tag <fieldset></fieldset>

- Dentro do <fieldset>, cada um dos campos deve ser colocado dentro de tags <p> para que possam ser formatados adequadamente na página

Para criar uma caixa de texto dentro dos campos do <fieldset>, é necessário seguir a estrutura do comando a seguir:

  • <p> Nome: <input type="text" name="nomeObjeto" id="idObejto" ></p>

Nesse exemplo, a tag <input> está sendo usada para criar uma caixa de texto, e os atributos "type", "name" e "id" estão sendo definidos para essa caixa de texto. Certifique-se de que a escrita esteja correta e os atributos estejam devidamente configurados de acordo com a necessidade do seu formulário

- Existem outros atributos que complementam a tag <input> e são: "size", "maxlength" e "placeholder". O primeiro determina quantos caracteres serão visíveis na caixa de texto. O segundo define o limite máximo de caracteres que podem ser digitados na caixa de texto. O terceiro cria um texto de orientação na caixa de texto, que é exibido de forma sombreada.

- Para criar botões de opção, utiliza-se o atributo "radio" no tipo (type) da tag <input>. Além disso, é possível implementar o atributo "checked", que automaticamente selecionará uma opção quando o usuário acessar o site.

- A tag <label>/label> é usada nos campos do formulário para criar rótulos associados aos elementos de entrada, como caixas de texto, botões de opção, entre outros. Quando o usuário clica no rótulo, o foco é automaticamente transferido para o elemento de entrada associado, como uma caixa de texto, tornando a experiência do usuário mais acessível e conveniente.

Ao criar um formulário, é necessário que os dados fornecidos nele sejam enviados para serem processados e armazenados em algum servidor, por exemplo. E para essa finalidade, existem dois métodos chamados "GET" e "POST" que são declarados na tag <form> do HTML. Basicamente, esses métodos são usados para enviar informações para o servidor. No entanto, existe uma diferença importante entre os dois métodos que deve ser levada em consideração:

- Usando o método get, os dados do formulário serão exibidos na URL como parâmetros de consulta. O get não é um método aconselhável em formulários que possuiram informações muito sensíveis como senhas, email, cpf e etc. Isto porque, todos estes dados ficarão exibidos na URL do site o que diminui a segurança. Este método é util, por exemplo, em formulários de pesquisa ou de opinião em que não é necessário inserir dados muito sensíveis.

- Usando o método post, os dados do formulário serão enviados no corpo do protocolo HTTP para o servidor. Para formulários que receberam dados sensíveis, o post é o mais aconselhável pois, possui mais segurança já que os dados são enviados de forma oculta diferente do que acontece no método get.

- Segue a estrutura abaixo para declarar o método post ou get na tag <form>:

  • <form method="post" id="id_metodo" action="mailto:exemplo@gmail.com">

OBS: o 'action' diz para onde os dados do formulário serão enviados. Os dados podem ser enviados à um servidor, podem ser enviados para um email ou para algum site (nesse caso, é necessário inserir a URL do site como valor para o action). No caso, a estrutura modelo de comando acima esta usando o valor mailto (que permite enviar os dados para um email).

A tag <select></select> é usada para criar um menu suspenso que permite ao usuário selecionar uma opção de resposta de uma lista de escolhas. Por exemplo, ao selecionar o estado em um campo 'Estado', será exibida uma lista suspensa com todos os estados territoriais, e o usuário poderá escolher em qual estado reside. Para usar essa estrutura, dentro da tag <select> você deve declarar as tags filhas chamadas <option>>/option>. É dentro dessas tags <option> que você coloca as opções de resposta que o usuário poderá escolher para o campo.

<input type="submit" value="Enviar"/> este input serve para criar um botão no formulário. Assim que o botão for clicado pelo usuário, os dados do formulário serão enviados.

Para chamar uma função JS no formulário, deve-se usar na tag <form> o comando oninput. Neste comando, basta passar apenas o nome da função no comando: oninput="nome_funcao()"
